# sem-funcoes-arrays

## ✍️ Descrição do projeto

Este repositório contém programas em JavaScript que resolvem exercícios de manipulação de vetores sem utilizar funções prontas de arrays.

Os objetivos são:
- Implementar algoritmos que usem estruturas de vetores para armazenar dados.
- Executar e validar os programas garantindo o funcionamento correto.
- Controlar versionamento local e remoto com Git.

## 🎯 Tarefas propostas

1) Faça um programa que leia 7 nomes de pessoas e guarde-os em um vetor. No final, mostre uma listagem com todos os nomes informados, na ordem inversa daquela em que eles foram informados. 7️⃣

2)  Faça um programa que leia um vetor com 8 números reais e calcule a média dos valores armazenados. Após isso, exiba os valores do vetor que estão acima da média. 8️⃣

3) Crie um programa que leia o nome e a idade de 9 pessoas e guarde esses valores em dois vetores, em posições relacionadas. No final, mostre uma listagem contendo apenas os dados das pessoas menores de idade (nome e idade). 9️⃣

4) Leia dois vetores de 5 elementos cada (valores inteiros). Crie um terceiro vetor que contenha apenas os elementos que estão em ambos os vetores (interseção, sem repetição). 5️⃣
   - Exemplo:
     - Vetor 1 → [1, 2, 3, 4, 5]
     - Vetor 2 → [3, 4, 5, 6, 7]
     - Resultado → [3, 4, 5]
